# Trabalho em Grupo – Apresentação dos Integrantes

Este repositório reúne as páginas individuais desenvolvidas pelos integrantes do grupo, com o objetivo de apresentar informações pessoais, curiosidades e viagens de cada menbro.

Cada membro criou sua própria página, contendo uma breve apresentação, interesses e habilidades. Os links para essas páginas estão organizados no documento index, presente no repositório. A fim de facilitar a navegação e o acesso ao perfil de cada integrante.

## Objetivo

O projeto tem como finalidade desenvolver habilidades em:

* Organização de projetos no GitHub
* Utilização do GitHub no terminal
* Trabalho em equipe e colaboração
